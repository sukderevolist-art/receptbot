# Telegram-бот для автопостинга рецептов

Бот каждый день по расписанию ищет рецепт на сайте «Едим дома», формирует короткий текстовый пост и публикует его в канал `@good_morning_you`.

На первом этапе бот публикует только текст, без картинок.

## Что умеет бот

- `/start` — проверка, что бот отвечает.
- `/test` — найти рецепт и отправить его в личный чат без публикации в канал.
- `/post_now` — сразу найти рецепт и опубликовать его в канал `@good_morning_you`.
- Автопостинг каждый день в 10:00 по часовому поясу из `.env`.
- Проверка повторов по ссылке на источник.
- Запись ошибок и публикаций в лог.

## Установка

Нужен Python 3.11 или новее.

Текущая версия бота запускается на стандартной библиотеке Python. Внешние зависимости оставлены в `requirements.txt` для следующего этапа, когда понадобится версия на `aiogram`.

На Windows можно сразу запускать `run.bat`. Если хотите создать виртуальное окружение, запустите `install.bat`.

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

На Linux или macOS активация окружения выглядит так:

```bash
source .venv/bin/activate
```

## Как создать Telegram-бота

1. Откройте Telegram.
2. Найдите `@BotFather`.
3. Отправьте команду `/newbot`.
4. Укажите имя и username бота.
5. BotFather выдаст токен. Его нужно вставить в `.env`.

## Как добавить бота в канал

1. Откройте канал `@good_morning_you`.
2. Перейдите в настройки канала.
3. Откройте раздел администраторов.
4. Добавьте созданного бота администратором.
5. Дайте ему право публиковать сообщения.

Без прав администратора бот не сможет отправлять посты в канал.

## Настройка `.env`

Скопируйте `.env.example` в `.env` и заполните токен:

```env
BOT_TOKEN=ВАШ_ТОКЕН_БОТА
CHANNEL_ID=@good_morning_you
POST_TIME=10:00
TIMEZONE=Europe/Moscow
```

Токен хранится только в `.env`. В Python-код его вставлять не нужно.

## Запуск

На Windows можно запустить `run.bat`.

```bash
python bot.py
```

Если запускаете вручную из PowerShell и видите странные символы в консоли, перед запуском можно включить UTF-8:

```powershell
$env:PYTHONUTF8=1
```

После запуска бот начнет принимать команды и включит ежедневное расписание.

## Проверка

1. Напишите боту `/start`.
2. Напишите `/test`, чтобы получить тестовый рецепт в личный чат.
3. Напишите `/post_now`, чтобы сразу опубликовать рецепт в канал.

Если публикация не прошла, проверьте, что бот добавлен администратором в `@good_morning_you` и имеет право отправлять сообщения.

## Запуск на сервере

1. Скопируйте папку проекта на сервер.
2. Установите Python 3.11+.
3. Создайте виртуальное окружение.
4. Установите зависимости.
5. Создайте `.env`.
6. Запустите `python bot.py`.

Для постоянной работы лучше запускать через `systemd`, `supervisor`, Docker или другой менеджер процессов.
